TROUBLESHOOTING QUICK REFERENCE

  → Re-discover plugins: sudo docker exec -i main_app_1 php artisan cms:plugin:discover

If logging not working:
  → Check directory permissions: sudo docker exec -i main_app_1 ls -la /var/www/html/storage/logs/
  → Check log channel config in RezgoConnectorServiceProvider.php boot() method

If database connection fails:
  → Verify MySQL is running: sudo docker ps | grep mysql
  → Check DB credentials in .env file

If orders don't submit:
  → Check rezgo_logs table: SELECT * FROM rezgo_logs ORDER BY created_at DESC;
  → Check mappings: SELECT * FROM rezgo_product_mappings WHERE is_active=1;
  → Verify sync_enabled: SELECT * FROM rezgo_settings WHERE setting_key='sync_enabled';
